MitM(Man in the Middle) implementation in LTE

  • Fake UE and Fake eNB: C++, C
  • MitM relay: Python

MitM environment was built in LTE network to verify vulnerabilities found through formal verification. In the case of Fake gNB and Fake UE, it was executed through srsRAN, and a program to relay messages between the two entities was made using python.

LTE Fuzzing

  • UE simulator: C++, C
  • Message Analyzer(CLI based): Python

We tried to find vulnerabilities by conducting fuzzing tests on LTE networks. The UE simulator for testing was implemented based on srsRAN, and the analyzer for analyzing the results was implemented in the form of Ubuntu CLI using python.

5G Core Network Testing

  • UE, gNB emulator: C, Docker, Kubernetes
  • 5G Core Network: C, Docker, Kubernetes
  • Packet forwarding: Python

From 5G, each network entity runs a container-based program. In the case of 5G Core Network, each network entity is implemented and managed using kubernetes. To test this, UE and gNB emulator were implemented using kubernetes, and the program for packet forwarding was made in python.
